#  DevCard — Next.js ID Generator
-------------------------------------

DevCard is an interactive digital card simulator built using Next.js. It allows developers to input their custom profile summaries and instantly compiles a clean, visually futuristic network ID identity badge complete with automated GitHub avatar integrations.

##  Key Concepts Learned
*  **Next.js Project Architecture:** Understanding the structure of the `src/app` framework layout routing.
*  **Client Components:** Using the `'use client'` directive to safely toggle user reactive interfaces.
*  **API Asset Resolution:** Rendering remote assets smoothly based on active text hooks.

## Running the App
    1. Install packages: `npm install`
    2. Run locally: `npm run dev`
    3. View on: `http://localhost:3000`
